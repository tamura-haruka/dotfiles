# dotfiles
my dotfiles

## component
* .bash_profile
* .bashrc
* .inputrc
* .latexmkrc
* .vimrc
* init.vim
* init.lua

## path
### mac/linux
.bash_profile        ->    ~
.bashrc              ->    ~
.inputrc             ->    ~
.latexmkrc           ->    ~
.vimrc               ->    ~
init.vim             ->    ~/.config/nvim/
init.lua             ->    ~/.comfig/nvim/

### windows
.latexmkrc           ->    ~
.vimrc               ->    ~
init.vim             ->    ~/AppData/Local/nvim/
init.lua             ->    ~/AppData/Local/nvim/

## note
.latexmkrc内のpdfビューアは、windows用のパスになっている。
init.vimはなるべく使わずにinit.luaを推奨する。
