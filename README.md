# dotfiles
my dotfiles

## component
* .bash_profil
* .bashrc
* .inputrc
* .latexmkrc
* .vimrc
* init.vim
* init.lua

## path
### mac/linux
.bash_profile        ->    ~<br>
.bashrc              ->    ~<br>
.inputrc             ->    ~<br>
.latexmkrc           ->    ~<br>
.vimrc               ->    ~<br>
init.vim             ->    ~/.config/nvim/<br>
init.lua             ->    ~/.comfig/nvim/<br>

### windows
.latexmkrc           ->    ~<br>
.vimrc               ->    ~<br>
init.vim             ->    ~/AppData/Local/nvim/<br>
init.lua             ->    ~/AppData/Local/nvim/<br>

## note
.latexmkrc内のpdfビューアは、windows用のパスになっている。<br>
init.vimはなるべく使わずにinit.luaを推奨する。<br>
